# tempo-memos

ブロックチェーン **Tempo** に関するメモ・調査ノートを置くリポジトリです。**日本人向け**を前提に、**README と記事本文は日本語** で運用します（X スペースなどでの話すネタ整理にも使う想定）。ルートの **llms.txt** のみ、ツール互換のため **英語** で書く。

## 参照リンク

- [Tempo](https://tempo.xyz/) — 決済向けに設計されたレイヤー1などの概要
- [Tempo on GitHub](https://github.com/tempoxyz) — 公式リポジトリ（クライアント、SDK、ドキュメントなど）

## 記事の置き方

- **置き場所は `docs/` 以下。** テーマごとに `docs/` の下でサブフォルダを切ってもよい（そのときもファイル名は内容ベース）。
- **本文は Markdown（`.md`）のみ。** 言語は **日本語** に統一する。
- **ファイル名は内容が推測できる単語**にする（例: `docs/subscriptions.md`、`docs/payment-lanes.md`）。日付だけのファイル名は使わない。

## llms.txt

ルートの **[llms.txt](llms.txt)** は本文を **英語** で書く（ツール・クローラとの互換のため）。内容はこのリポジトリの要約と、**日本語記事（`.md`）へのリンク一覧**（`docs/` 以下の相対パス）。記事を追加したら **llms.txt の一覧も更新する**。

## エージェント（AI）向け

編集するエージェントは **[AGENTS.md](AGENTS.md)** を読む（チェックリスト・**作業で得た気づきの追記場所**はこちら）。
